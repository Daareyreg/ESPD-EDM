usageNote-en NOTE
Details:
Requirement is a generic class representing any type of prerequisite that may be desired, needed or imposed as an obligation.

CCCEV recommends to not use the Requirement class directly, but rather a more semanticall-enriched subclass such as Criterion, Information Requirement or Constraint. Also note that the Requirement class is specified at a more abstract level and is not to be used as the instantiation of a Requirement for a specific Agent.
