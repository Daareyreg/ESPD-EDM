=== 1.4 UBL-2.3 and TOOP Requirements

During the development of v3.0.0 some requirements arose from TOOP related to the Evidence class. The aforementioned requirements that can be found in the following GitHub https://github.com/ESPD/ESPD-EDM/issues/277[issue], are fully mappable to the UBL 2.3 Schema.

To check the mapping, please access to these https://github.com/ESPD/ESPD-EDM/tree/3.0.0/docs/src/main/asciidoc/assets/[Excel file] or https://github.com/ESPD/ESPD-EDM/tree/3.0.0/docs/src/main/asciidoc/assets/[ODS file].
