== 1. UBL Based and distribution package

Semantic assets have been defined by link:https://joinup.ec.europa.eu/category/glossary/semantic-interoperability-asset[ISA] as "a collection of highly reusable metadata (e.g. xml schemata, generic data models) and reference data (e.g. code lists, taxonomies, dictionaries, vocabularies) which are used for e-Government system development".
